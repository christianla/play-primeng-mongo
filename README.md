# play-primeng-mongo
This is a simple demo app with Scala Play Framework, PrimeNG/Angular and mongodb connection.

## description
Server and client support CRUD-Operations for cars (based on PrimeNG showcase).

## 

base technologies:

- scala 2.11.11
- Play Framework 2.5
- Angular 2 (Using Typescript)
- PrimeNG
- simple-mongo
